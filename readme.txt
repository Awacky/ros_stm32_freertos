该分支为发布的STM32F407分支
该分支支持:A4950 BTN797x ESC DCM_FOC等电机驱动

工程架构分为bootloader和app两个工程
bootloader的地址为:0x8000000~0x8008000
App的地址为:0x8010000~0x8FFFFFF
使用前请先全片擦除flash,然后把xx/Bootloader/StarsRobotBoot_xx.hex文件烧入flash中
打开工程编译并下载App工程,只要在之后的程序下载中,没有改变APP的地址
并且没有全片擦除则无需重复下载bootloader(),只需要按正常的keil工程进行代码烧入即可.
需要修改自己的硬件定义则只需要在STM32Bsp\HwConfig里建立自己的硬件即可,可拷贝原有的进行端口等硬件修改
APP工程是根据芯片ID进行绑定的,如果更换了芯片或者开发板,需要联系卖家进行授权.授权一次终身有效.
